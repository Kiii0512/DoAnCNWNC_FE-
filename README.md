## Feature: Khuyến mãi (Promo)
Đã triển khai trong thư mục `promoList/`:
- `promoList/promoList.html` — danh sách chương trình khuyến mãi
- `promoList/promoDetail.html` — trang chi tiết chương trình
- `promoList/promo.css` — style cho promo
- `promoList/promo.js` — load dữ liệu và render list & detail
- `promoList/promos.json` — sample data

### Xem thử local
- Mở `promoList/promoList.html` bằng Live Server hoặc `python -m http.server` trên root rồi truy cập `http://localhost:8000/promoList/promoList.html`.
